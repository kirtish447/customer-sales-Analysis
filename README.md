# Customer Sales & Behavior Analysis

## 📌 Project Overview
This project focuses on analyzing customer behavior and sales data to generate actionable business insights.

## 🎯 Problem Statement
The objective of this project is to analyze sales and customer data to identify revenue drivers, improve customer retention, and optimize product strategy.

## 📊 Key KPIs
- Revenue
- Number of Orders
- Average Order Value (AOV)
- Customer Retention Rate
- Repeat Purchase Rate

## ❓ Business Questions
- Which products generate the most revenue?
- Who are the most valuable customers?
- What are the peak sales periods?
- What is the repeat purchase behavior?
- How can we improve retention?

## 🛠 Tools & Technologies
- SQL
- Excel
- Power BI
- Python (Pandas)

## 📁 Project Structure
- data/raw → Raw dataset
- data/processed → Cleaned data
- sql → SQL queries
- notebooks → Python analysis
- dashboard → Power BI files
- reports → Final insights
